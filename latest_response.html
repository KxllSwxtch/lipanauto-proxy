Таможенный калькулятор – расчет таможенных платежей при ввозе автомобиля

Таможенный калькулятор, расположенный на этой странице, позволяет быстро рассчитать таможенные платежи при ввозе и таможенном оформлении транспортных средств: легкового автомобиля, грузовика, мотоцикла, автобуса и иных авто- мототранспортных средств, прицепов и водных судов для личного пользования или для юридических лиц.

Для расчета платежей по другим товарам, ввозимым юридическими лицами, рекомендуем Универсальный таможенный калькулятор calc.tks.ru.

Для расчета платежей при ввозе, пересылке, доставке транспортной компанией иных товаров для личного пользования физическими лицами, пользуйтесь Калькулятором Расчета т/п для физлиц.

Результатом расчета являются суммы платежей, рассчитанные при ввозе авто:

  • для личного пользования по Единым ставкам, утвержденным Таблицей 2 Приложения 2 к Решение Совета ЕЭК от 20.12.2017 N 107.
  • для юридического лица по ставкам, установленным Единым таможенным тарифом (ЕТТ) ЕАЭС, утвержденным Решением Совета ЕЭК от 14.09.2021 года N 80.

Расчет таможенных платежей при ввозе автомобиля для личного пользования физических лиц выполняется с применением единой ставки – льготного вида таможенного платежа в виде фиксированной ставки, величина которой зависит от объема двигателя (для б/у автомобилей) или стоимости (для новых автомобилей).

ВНИМАНИЕ! С 1 апреля 2024 года изменился порядок расчета утилизационного сбора при ввозе в РФ автомобилей, растаможенных на территории ЕАЭС по декларации на товары (ДТ).

Для корректного расчета необходимо использовать информацию из таможенных документов, оформленных при декларировании транспортных средств в государстве ЕАЭС (пошлина, акциз, НДС из ДТ), а также из Перечня сведений ФТС России для целей исчисления утилизационного сбора, размещенного на официальном сайте Минпромторга (средняя стоимость автомобиля в рублях с учетом модели и марки).
